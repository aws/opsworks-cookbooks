v1.30 - 2/3/2015
-------
- Update to latest JDKs for 6 and 7. JDK7 will be EOL April 2015
- Fix up Travis support.
- Add ability to install JCE policy files for oracle JDK #228
- Change connect timeout to 30 seconds
